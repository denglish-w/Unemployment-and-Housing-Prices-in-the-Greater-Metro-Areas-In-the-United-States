



# Data Sources
All data sources will be found in the 'data' folder. 

 1. hpi_po_metro.csv is a dataset from the Federal Housing Finance Agency. This dataset shows the seasonally adjusted and not seasonally adjusted index values.
 https://www.fhfa.gov/data/hpi/datasets?tab=quarterly-data (download from the "100 Largest Metropolitan Statistical Areas (Seasonally Adjusted and Not Adjusted)")


 2. ssamatab1.csv is a dataset from the Bureau of Labor Statistics. It is a smooth, seasonally adjusted estimate of the unemployment rate in major metro areas.
 https://www.bls.gov/lau/metrossa.htm
